Open Design 라우트나 daemon 없이 현재 폴더에서 실행됩니다. 상품 입력을 하나의 실제 프롬프트로 만들고 선택한 CLI 또는 BYOK HTTP provider에 전달합니다.

상품 입력

상세페이지 또는 광고 세트 생성을 위해 상품명, 설명, 요구사항, 이미지/자료, 목표 마켓을 입력합니다.

prompt.contract
생성 모드

광고 자동화 입력

브랜드 URL은 서버에서 query, fragment, 자격 정보를 제외한 참조 주소로만 정규화합니다. Phase 1에서는 외부 페이지를 가져오지 않습니다.

ad-set

광고 세트 모드는 16개 카피 앵글 중 상품/마켓/무드에 맞는 5개를 자동 추천하고, 결과 갤러리와 export payload에 포함합니다.

입력한 내용은 상세페이지, 광고 세트, 이미지 생성 프롬프트에 반드시 반영해야 할 조건으로 포함됩니다.

이미지/자료
이미지/자료를 여기에 드래그앤드랍하거나 파일 업로드 버튼을 누르세요 png, jpg, jpeg, webp, pdf, txt, md, csv 파일을 여러 개 첨부할 수 있습니다.

    로컬 실제 경로는 저장하지 않고 파일명, 타입, 크기, 가능한 텍스트 요약과 미리보기 가능 여부만 생성 입력에 포함합니다.
