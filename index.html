            무드 프리셋 도움말

            무드 프리셋은 광고 말투를 고르는 버튼입니다. 같은 상품이라도 광고의 첫 문장, 설명 방식, 강조점이 달라집니다.

            • 처음이면Clean
            • 클릭 유도Bold
            • 고급 상품Premium
            • 과일/채소Fresh
            • 기능 강조Technical
            • 할인/행사Energetic
            Clean · 정돈된 정보형
            깔끔하고 차분하게 핵심 정보만 전달합니다. 생활용품, 전자제품, 사무용품처럼 정보가 중요한 상품에 좋습니다.
            Bold · 강한 첫 문장
            첫 줄부터 눈길을 끄는 강한 문장으로 시작합니다. SNS 광고, 배너, 클릭 유도 광고에 좋습니다.
            Editorial · 설명형 무드
            잡지나 블로그처럼 차근차근 설명합니다. 브랜드 스토리, 사용 장면, 감성 설명에 좋습니다.
            Premium · 고급 신뢰형
            비싸 보이고 믿음직한 톤으로 말합니다. 화장품, 인테리어, 선물용, 고가 상품에 좋습니다.
            Warm · 따뜻한 생활형
            친근하고 부드러운 생활감으로 말합니다. 가족, 집, 육아, 생활용품처럼 일상에 가까운 상품에 좋습니다.
            Fresh · 과일/채소 신선형
            수확한 듯한 신선함, 산지, 식감, 보관 포인트를 쉽게 말합니다. 과일, 채소, 샐러드, 제철 식품 광고에 좋습니다.
            Minimal · 절제된 제품형
            말을 줄이고 제품 자체를 돋보이게 합니다. 디자인 제품, 미니멀 브랜드, 깔끔한 이미지 광고에 좋습니다.
            Energetic · 활기 있는 프로모션형
            밝고 빠른 행사 느낌을 만듭니다. 할인, 이벤트, 신상품 출시, 시즌 프로모션에 좋습니다.
            Technical · 스펙 검증형
            감성보다 기능, 수치, 성능을 강조합니다. 전자기기, 도구, 기능성 제품에 좋습니다.
            Gift · 선물 제안형
            누구에게 선물하면 좋은지 중심으로 말합니다. 명절, 생일, 집들이, 기념일 상품에 좋습니다.
            Seasonal · 시즌/상황형
            계절이나 상황에 맞춰 말합니다. 여름, 겨울, 새 학기, 명절, 여행, 이사 시즌에 좋습니다.

            How to use

            생성 전에 준비할 것과 앱 흐름을 한 번에 확인합니다.

            before.start

            사용 전 준비

            생성 버튼을 누르기 전에 아래 4가지를 먼저 확인하세요.

            1. AI 로그인 사용할 AI에 로그인하거나 BYOK API 키를 준비합니다.
            2. 엔진 설정 로컬 CLI/BYOK, 모델, timeout을 확인합니다.
            3. Preflight 통과해야 AI 엔진과 연결됩니다. 실패하면 설정을 고칩니다.
            4. 상품 자료 상품명, 설명, 사진, 목표 마켓을 준비합니다.

            이미지 큰 화면

            생성 이미지를 크게 확인하고 선택한 이미지만 추가 수정합니다.

            수정 실행은 선택한 이미지 1장만 reference로 사용해 새 수정본을 생성합니다. 원본 파일은 덮어쓰지 않습니다.
