Open Design 라우트나 daemon 없이 현재 폴더에서 실행됩니다. 상품 입력을 하나의 실제 프롬프트로 만들고 선택한 CLI 또는 BYOK HTTP provider에 전달합니다.

로컬 앱 준비

상품 입력

생성 엔진으로 전달할 상품명, 설명, 요구사항, 이미지/자료, 목표 마켓을 입력합니다.

prompt.contract
이미지/자료
이미지/자료를 여기에 드래그앤드랍하거나 파일 업로드 버튼을 누르세요 png, jpg, jpeg, webp, pdf, txt, md, csv 파일을 여러 개 첨부할 수 있습니다.

    로컬 실제 경로는 저장하지 않고 파일명, 타입, 크기, 가능한 텍스트 요약과 미리보기 가능 여부만 생성 입력에 포함합니다.

    이미지 생성 옵션

    설정에서 Codex CLI ImageGen을 켜면 이 옵션으로 실제 이미지 파일을 생성합니다.
